# Email from Claude Team about Claude Fable 5


On Tue, Jun 9, 2026 at 1:00 PM Claude Team <no-reply@email.claude.com> wrote:
Claude

Today, we're introducing Claude Fable 5, our next generation of intelligence for agents, coding, and the hardest knowledge work.

Here's what's new on the Claude Platform (API):

Claude Fable 5: Works independently for longer than any prior generally available Claude model, planning across stages and verifying its own work along the way. Available at $10 input / $50 output per million tokens.

Claude Managed Agents: Pair Claude Fable 5 with Claude Managed Agents, our harness for long-running agentic work, now in public beta. Claude Fable 5 works out of the gate with no changes needed.

Advisor strategy: Claude Fable 5 is available as an advisor model. Faster, lower-cost worker models can call Claude Fable 5 mid-task to check their plan and evaluate their work, leading to improved performance.

Safety classifiers: Claude Fable 5 blocks a set of cybersecurity and biology requests. Blocked requests return an error rather than a model response, and you aren't charged for them.

Fallback API: Opt in to fallback on the Messages API to continue blocked requests on Claude Opus 4.8 at Opus pricing or easily roll your own fallback with Anthropic SDK support. In Managed Agents, fallback is built in.

Data retention requirements: Claude Fable 5 requires a limited 30 day retention period. Retained data is used only to detect and prevent serious misuse, and is not used to train Claude. Admins must accept updated terms in the Claude Console before using the model. See our help center to learn more.

Read the blog
Claude Fable 5 is available on our API, Amazon Web Services, Google Cloud, and Microsoft Foundry. To upgrade existing workloads, you can use the claude-api skill in Claude Code to update model names, prompts, and settings by running /claude-api migrate.
